# Template Trong C++

## Template trong C++ là gì?
- Template (khuôn mẫu) là một từ khóa trong C++, và là kiểu dữ liệu trừu tượng tổng quát hóa cho các kiểu dữ liệu `int`, `float`, `double`, `bool`, ...

- Có 2 loại là funtion template và class template.

- Template giúp người lập trình định nghĩa tổng quát cho hàm và lớp thay vì phải nạp chồng (overloading) cho từng hàm hay phương thức với những kiểu dữ liệu khác nhau.

## Cách sử dụng template đối với hàm
### Syntax 
```c
template <class T>
T someFunction(T arg1, ...)
{
   ...
}
```

Liên hệ qua hàm swapval sau đây để thấy điều đó.
```c
template <class val>
void swapVal(val &a , val &b)
{     
     val temp;
     temp = a;
     a = b;
     b = temp;
}
```

Thay vì đưa kiểu dữ liệu biến là int hay float, ... thì nó làm là val sẽ cho phép truyền đa dạng dữ liệu không cần nạp chồng.

Một template `<class temp>` , temp chỉ có tác vụ định nghĩa cho 1 kiểu dữ liệu duy nhất do đó chúng ta không thể gán 1 biến kiểu int và double cho cùng 1 kiểu dữ liệu temp trong cùng một thời điểm.

**Ví dụ:**
```c
#include <bits/stdc++.h>
using namespace std;
template <class temp>
void sum(temp intVal , temp doubleVal) 
{
	cout<<int1+double1;
}
int main()
{
	int intVal = 1;
	double doubleVal = 1.1;
	sum(intVal , doubleVal);
 return 0;
 }
```

Để giải quyết vấn đề đó thì ta khai báo thêm một kiểu dữ liệu tổng quát vào như sau:

```c
#include <bits/stdc++.h>
using namespace std;

template <class temp , class temp1>
void sum(temp intVal , temp1 doubleVal) 
{
	cout<<int1+double1;
}

int main()
{
	int intVal = 1;
	double doubleVal = 1.1;
	sum(intVal , doubleVal);
 return 0;
 }
```

khi viết hàm cũng vậy, khi lấy temp làm kiểu dữ liệu trả về cho hàm, chúng cũng chỉ nhận kiểu dữ liệu của biến trả về theo kiểu dữ liệu của biến truyền vào mà không trả về một kiểu dữ liệu khác được.

```c
#include <bits/stdc++.h>
using namespace std;
template <class temp>
temp thuong(temp x1, temp x2)
{
	return (x1*1.0)/x2;
}
int main()
{
	int x1 = 1 , x2 = 2;
	cout<<thuong(x1,x2);
 	return 0;
 }
```
