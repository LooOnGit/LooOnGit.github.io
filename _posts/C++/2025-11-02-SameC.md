# C Features That Remain the Same in C++

## Try catch throw

Dùng trong xử lí ngoại lệ, tức là bắt lỗi và in ra lỗi cho ta biết, để từ đó ta có thể sửa lại input bài toán.

```c++
#include <iostream>
using namespace std;

int main() {
    try {
        int a = 10;
        int b = 0;

        if (b == 0) {
            throw "Division by zero!";  // Throw an exception
        }

        cout << "Result: " << a / b << endl;
    }
    catch (const char* msg) {  // Catch string type exception
        cout << "Error: " << msg << endl;
    }

    cout << "Program continues..." << endl;
    return 0;
}

```
